# 프로젝트 소개


📱👤</br>


보이스피싱 통화 내용을 음성을 분석하여 얻은 보이스피싱 빈출 단어 데이터베이스를 활용하여 보이스피싱 여부를 판단하는 애플리케이션이다.</br>

사용자가 희망하는 통화 녹음 파일을 받아 애플리케이션 내에서 해당 음성 파일의 텍스트를 분석하여 보이스피싱 여부를 판단하고 보이스피싱으로 판단되었을 때, 사용자에게 대처 방법을 안내한다.

</br>
